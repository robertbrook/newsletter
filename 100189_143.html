http://blog.sanctum.geek.nz/advanced-vim-registers/
When you cat and paste in Vim, you can use one of 26 'named registers' to store and manipulate the text. You can add text rather than just deleting the old stuff, look over several registers together, paste into a register that you can't read from and much more. Some automatic registers store the current file name, last used command and so on.
