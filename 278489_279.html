A correspondent asks: "Is there some clear benefit in using Dropbox instead of emailing a document?" - which I said I'd try to write something like an answer to. Most often, people will use Dropbox instead of attaching large files to an email. Things in Dropbox can be shared with a URL, which you can give to several people, if you need to. If your attachment is small, you might as well stick with email.
